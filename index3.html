Le ❬ p ❭ en HTML

Les paragraphes sont des éléments blocs.

Pour modifier l'espacement entre les paragraphes, il faudra utiliser la propriété CSS margin.

Il ne faut pas insérer de paragraphes vides ou d'éléments < br > afin de créer un espace.

L'utilisation de paragraphes vides (des éléments HTML ❬ p ❭ sans contenu) est problématique pour les personnes qui naviguent sur une page à l'aide d'outils d'assistance.

Les lecteurs d'écran, par exemple, pourraient annoncer l'élément mais pas le contenu associé ce qui peut être frustrant ou source de confusion.

L'élément de rupture thématique

❬ hr ❭ représente un changement thématique entre des éléments de paragraphe.

Par exemple: un changement de décor dans un récit, un changement de sujet au sein d'une section.

L'élément de saut de ligne

❬ br ❭ crée un saut de ligne (un retour chariot) dans le texte
Il s'avère utile lorsque les sauts de ligne ont une importance
Par exemple lorsqu'on écrit une adresse ou un poème
Cet élément est inclus à chaque point où nous voulons que le texte soit interrompu
Le texte après le ❬ br ❭ recommence au début de la ligne suivante du bloc de texte
Rappel: ❬ br ❭ ne doit pas être utilisé de façon abusive afin d'augmenter l'espace entre les lignes d'un texte
Pour cela, on utilisera la propriété CSS margin sur l'élément HTML < p >
Il n'a aucun rôle purement stylistique et il sert uniquement à créer une rupture de ligne au sein d'un bloc de texte
Cet élément pourra très peu être mis en forme avec CSS
Utiliser une suite d'éléments ❬ br ❭ pour uniquement rajouter un espace entre des paragraphes
Sera source de confusion ou de frustration pour les personnes qui utilisent des outils d'assistance
Les lecteurs d'écran, par exemple, pourront annoncer la présence de l'élément alors qu'il n'y a aucun contenu associé
Encore une fois: espace supplémentaire = margin, la propriété CSS

L'élément de division du contenu

❬ div ❭ (ou division) n'a aucun effet sur le contenu ou la mise en page tant qu'il n'est pas mis en forme d'une manière quelconque à l'aide de CSS.

En tant que conteneur « pur », il ne représente rien en soi. Il est plutôt utilisé pour regrouper le contenu afin qu'il puisse être facilement stylé à l'aide des attributs "class" ou "id", pour marquer une SECTION d'un document comme étant écrite dans une langue différente (à l'aide de l'attribut "lang"), etc.

L'élément ❬ div ❭ doit uniquement être utilisé lorsqu'il n'existe aucun autre élément dont la sémantique permet de représenter le contenu (par exemple < article > ou < nav >).

Cet exemple crée une boîte avec une ombre en appliquant la mise en forme via du CSS sur l'élément ❬ div ❭!

Le ❬ span ❭ en HTML

Il est un conteneur générique en ligne (inline) pour les contenus phrasés.
Comme pour < div >, il peut être utilisé pour grouper des éléments afin de les mettre en forme (grâce aux attributs "class" ou "id" et aux règles CSS) ou parce qu'ils partagent certaines valeurs d'attribut comme "lang".
❬ span ❭ est très proche de l'élément < div >, mais l'élément < div > est un élément de bloc, alors que ❬ span ❭ est un élément en ligne
Il doit uniquement être utilisé lorsqu'aucun autre élément sémantique n'est approprié.

L'élément de liste de descriptions

❬ dl ❭ représente une liste de descriptions sous la forme d'une liste de paires associant des termes (fournis par des éléments < dt >) et leurs descriptions ou définitions (fournies par des éléments < dd >).

On utilisera par exemple cet élément pour implémenter un glossaire :

< dt >
L'élément pour le terme d'une description
Identifie un terme dans une liste de définitions ou de descriptions.
Plusieurs éléments qui se suivent indiqueront qu'ils partagent la définition/description fournie par le prochain élément < dd >.
< dd >
L'élément de détail d'une description
Fournit la description, la définition ou la valeur du terme précédent dans une liste de description.
